# StayFlix Analytics Pro

**Plateforme d'intelligence analytique Netflix × Hôtel | ENSEA 2025-2026**

## Démarrage rapide

```bash
pip install -r requirements.txt
python run.py
# Ouvrir http://localhost:5000
```

## Comptes de démonstration

| Rôle | Username | Mot de passe | Accès |
|------|----------|-------------|-------|
| Admin | admin | AS3admin2026 | Complet |
| Manager | manager | Manager@2026 | Complet + Export |
| Analyste | analyst | Analyst@2026 | Tout sauf export |
| Viewer | viewer | Viewer@2026 | Consultation |

## Fonctionnalités

- **Netflix** : Dashboard (filtres dynamiques), Contenu & Durées, Ratings & Audience, Timeline, Explorateur avancé
- **Hôtel** : Dashboard (filtres dynamiques), Réservations, Revenus & ADR, Annulations, **Carte Mondiale interactive (Leaflet)**, Explorateur avancé
- **Animations** : Chiffres animés, graphiques dynamiques, ticker live
- **Filtres temps réel** : Tous les dashboards se mettent à jour sans rechargement
- **Export Excel** : Admin/Manager uniquement
- **Carte mondiale** : 35+ pays avec popups détaillés (ADR, annulations, revenus)
